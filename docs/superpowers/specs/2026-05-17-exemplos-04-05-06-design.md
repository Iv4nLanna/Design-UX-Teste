## Princípios desta rodada

Todos os três exemplos partem do zero (`starting_point: scratch`). Foco em:
- **Interação rica** — mouse, teclado, clique com resposta visual imediata
- **Criatividade pura** — sem skill ou referência externa; medir criatividade nativa do Claude
- **Detalhes profundos** — nenhum placeholder, cada camada visual é intencional

## 04 — Vaporwave / Synthwave

### Frontmatter

```yaml
title: Vaporwave / Synthwave
date: 2026-05-17
style: vaporwave
tags: [retro, neon, synthwave, parallax]
techniques: [css-3d, clip-path, web-audio, css-animation, custom-properties]
process_pattern: from-scratch
starting_point: scratch
ai_role: co-designer
iterations: 3
inspiration:
  - Estética visual dos anos 80
  - Capas de álbuns synthwave
difficulty: 3/5
```

### Conceito

Página de artista fictício "NEON PARALLAX". Grade 3D perspectiva com scroll infinito, céu degradê roxo→magenta→laranja, sol semicírculo pulsando com neon. Retro-futurismo dos anos 80 como linguagem visual completa.

### Estrutura visual

- **Fundo**: degradê vertical `#0d0015` → `#1a0030` → `#4a0080`
- **Céu**: gradiente `#ff6ad5` → `#c774e8` → `#ff9a3c` na metade superior
- **Sol**: semicírculo com `box-shadow` em camadas neon pulsando, listras horizontais via `repeating-linear-gradient`
- **Grade 3D**: `perspective` + `rotateX(75deg)` + `translateZ` animation loop infinito
- **Shapes**: pirâmides e losangos via `clip-path polygon`, rotacionando com `rotateX/Y`
- **Scanlines**: `repeating-linear-gradient` com `mix-blend-mode: overlay`, opacidade 0.15
- **Estrelas**: 100+ pontos via `box-shadow` em pseudo-elemento `::before`, zero JS

### Paleta

```css
--neon-pink: #ff6ad5;
--neon-cyan: #00f5ff;
--neon-orange: #ff9a3c;
--deep-purple: #0d0015;
--mid-purple: #4a0080;
--neon-yellow: #f5ff00;
```

### Interações

| Trigger | Resposta |
|---|---|
| Mouse move | Paralaxe em 3 camadas: grade (lento), shapes (médio), título (rápido) |
| Scroll | Aumenta velocidade da grade + revela seções com fade-in |
| Hover no título | Glitch cromático: spans R/G/B sobrepostos com offset animado `steps(1)` |
| Clique em qualquer lugar | Nota synth via Web Audio API (oscilador sawtooth, envelope rápido ~200ms) |

### Detalhes técnicos

- **Aberração cromática**: título composto de 3 `<span>` sobrepostos (vermelho, ciano, branco) com `transform: translate` animados independentemente
- **Paralaxe**: `mousemove` listener calcula offset normalizado -1→1, aplica `transform: translateX/Y` por camada com multiplicadores diferentes
- **Web Audio**: `AudioContext`, oscilador `sawtooth`, `GainNode` com `exponentialRampToValueAtTime`, frequências aleatórias na escala pentatônica (para sempre soar bem)
- **Grade**: elemento `div` com `transform: perspective(500px) rotateX(75deg)`, background `repeating-linear-gradient` criando linhas, animação `translateY` contínua

## 05 — Generativo com p5.js

### Frontmatter

```yaml
title: Campo de Fluxo Generativo
date: 2026-05-17
style: generative
tags: [generative-art, procedural, perlin-noise, interactive]
techniques: [p5js, perlin-noise, particle-system, canvas]
process_pattern: from-scratch
starting_point: scratch
ai_role: co-designer
iterations: 3
inspiration:
  - Campos vetoriais matemáticos
  - Arte generativa de Vera Molnár
difficulty: 3/5
```

### Conceito

Campo de fluxo vetorial com Perlin noise. Canvas fullscreen. Linhas nascem nas bordas, seguem vetores calculados pelo noise, acumulam trilha com fade lento — criando "fósseis" de movimento. O visitante está dentro do pensamento.

### Algoritmo central

```
para cada frame:
  desenha background com alpha baixo (8/255) → fósseis aparecem
  para cada partícula ativa:
    calcula ângulo = noise(x*scale, y*scale, tempo*0.003) * TWO_PI * 2
    adiciona vetor de velocidade baseado no ângulo
    adiciona atração suave para posição do mouse
    desenha linha do ponto anterior ao atual
    se saiu da tela ou atingiu max_age → respawn aleatório
```

### Paletas (tecla C para alternar)

| Nome | Cores |
|---|---|
| Oceano | `#001220`, `#0077b6`, `#00b4d8`, `#90e0ef`, `#caf0f8` |
| Fogo | `#03071e`, `#e85d04`, `#f48c06`, `#faa307`, `#ffba08` |
| Floresta | `#081c15`, `#1b4332`, `#40916c`, `#74c69d`, `#d8f3dc` |
| Aurora | `#10002b`, `#7b2d8b`, `#00f5d4`, `#fee440`, `#ffffff` |
| Mono | `#000000`, `#1a1a1a`, `#4d4d4d`, `#b3b3b3`, `#ffffff` |

### Interações

| Trigger | Resposta |
|---|---|
| Mouse move | Reposiciona atrator — todas as linhas curvam em direção ao cursor |
| Clique | Explosion de 200 partículas do ponto clicado, velocidade decaindo com `lerp` |
| Tecla `C` | Alterna paleta (cicla entre as 5) com fade suave |
| Tecla `R` | Reset com novo seed de noise — layout completamente diferente |
| Tecla `Space` | Pausa/retoma — congela o momento atual |

### Detalhes técnicos

- **Espessura variável**: `strokeWeight` de 0.2 a 2.0 baseado no gradiente local do noise
- **Opacidade variável**: partículas mais "rápidas" (vetor maior) são mais opacas
- **Dois octaves de noise**: escala grande (padrões globais) + escala fina (detalhes locais) somados com pesos 0.7 / 0.3
- **HSL shift temporal**: hue da paleta desloca +0.1° por frame — a paleta "respira" ao longo do tempo
- **Atração ao mouse**: força proporcional a `1/distance²`, limitada por distância mínima (evita singularidade)
- **Partículas**: pool de 800 partículas ativas simultâneas; idade máxima aleatória entre 100-400 frames

## 06 — Renascimento Digital

### Frontmatter

```yaml
title: Renascimento Digital
date: 2026-05-17
style: renaissance
tags: [classical, museum, generative-art, css-art]
techniques: [svg, css-art, filter-svg, clip-path, web-fonts]
process_pattern: from-scratch
starting_point: scratch
ai_role: co-designer
iterations: 3
inspiration:
  - Pintura renascentista italiana (séc. XV–XVI)
  - Rembrandt (chiaroscuro)
  - Fibonacci / Proporção Áurea
difficulty: 4/5
```

### Conceito

Galeria de 3 obras geradas em CSS/SVG puro — sem imagens externas. Arte clássica reinterpretada como código. Cada "tela" é uma composição visual construída com primitivas SVG, gradientes e filtros.

### As três obras

#### Obra I — "Proporção Áurea" (Spirale Aurea)
- Espiral de Fibonacci construída em SVG: sequência de arcos `<path>` com raios na proporção 1.618
- Paleta: dourado `#c9a84c`, marfim `#f5e6d3`, sépia `#8b6914`
- Fundo: gradiente radial escuro com textura pergaminho via `feTurbulence`
- Moldura: retangular com frisos geométricos — meandros gregos em SVG path

#### Obra II — "Retrato em Chiaroscuro" (Ritratto)
- Rosto geométrico: elipse para cabeça, triângulos para nariz, semicírculos para olhos, curvas bézier para boca
- Iluminação Rembrandt: gradiente radial vindo do canto superior esquerdo, sombra densa no lado direito
- Paleta: ocre `#c8a96e`, sombra `#1a0a00`, luz `#fff8e7`
- Moldura: oval com ornamentos de folha de acanto em SVG

#### Obra III — "Paisagem Clássica" (Paesaggio)
- Montanhas em 4 camadas com opacidade decrescente (perspectiva atmosférica)
- Céu com nuvens: `feTurbulence` + `feColorMatrix` para textura orgânica
- Coluna dórica no primeiro plano: `clip-path polygon` + capitéis em SVG
- Reflexo em água: segunda cópia das montanhas invertida com opacity 0.4 + `blur`
- Moldura: retangular com pilastras laterais em SVG

### Interações

| Trigger | Resposta |
|---|---|
| Hover numa obra | Zoom lento `scale(1.08, 600ms)` + spotlight radial que segue o mouse dentro da tela |
| Clique numa obra | Modo exame: expande para 80vw + painel lateral com título, técnica, "ano", descrição em latim e português |
| Mouse dentro da obra | Cursor muda para lupa (`cursor: url(...)`) — SVG inline de lupa |
| Scroll | Cortina de veludo desce revelando cada obra: `clip-path: inset(100% 0 0 0)` → `inset(0)` |
| Hover na vela | Chama aumenta + glow se intensifica |

### Detalhes técnicos

- **Pergaminho**: SVG filter inline com `feTurbulence baseFrequency="0.65" numOctaves="3"` + `feColorMatrix` para tom sépia + `feComposite` para mesclar com o fundo
- **Molduras SVG**: cada moldura é um `<svg>` único com `<path>` gerados — folhas de acanto no oval, meandros no retangular, pilastras no paisagístico
- **Modo exame**: painel usa `position: fixed`, entrada com `transform: translateX(100%)` → `translateX(0)` (300ms ease-out), fundo com `backdrop-filter: blur(4px)`
- **Vela animada**: `clip-path` para formato de chama, `box-shadow` com múltiplas camadas neon âmbar, `animation: flicker` com `steps(1)` e keyframes irregulares
- **Tipografia**: Cinzel Decorative (títulos de obras), EB Garamond italic (legendas e descrições latinas), ambas via Google Fonts
- **Placas de bronze**: `background: linear-gradient` dourado, `border-radius`, `box-shadow` inset para relevo, texto em Cinzel com `letter-spacing` generoso
